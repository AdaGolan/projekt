Zbuduj klasyfikator przewidujący, czy klient portugalskiej instytucji bankowej zapisze się na lokałę terminową ("y" = yes / no), na podstawie danych pochodzących z kampanii marketingowych prowadzonych telefonicznie. Wykorzystaj zbiór bank-additional.csv, zawierający 4119 przykładów oraz 20 cech wejściowych. Kampanie obejmowały wielokrotne kontakty telefoniczne z tymi samymi klientami, mające na celu ustalenie, czy produkt zostanie przez nich ostatecznie zakupiony. https://archive.ics.uci.edu/dataset/222/bank+marketing
